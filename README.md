# Evaluación Parcial Definitiva (Grupo 3)
Aplicación: Una página web de un restaurante, con pantallas para el cliente (para reservar) y el administrador (para administrar las reservas).

Integrantes:
- Caballero Crisostomo, Enzo Gabriel
- De la Cruz Elias, David Sebastian
- Quevedo Briceño, Juan Sebastian

Páginas realizadas hasta ahora:
- Register
- Login
- Pagina principal de administrador

Ejecución:
1. Hacer click al botón "Code"
2. Seleccionar "Codespaces" y presionar el símbolo "+".
3. Demorará un poco en cargar los archivos.
4. Una vez terminada la carga, ir a la consola y escribir "npm start".
5. Si no corre, eliminar node_modules y volver a hacer "npm install". Posteriormente, volver a ejecutar el comando "npm start".
